Web Application Development

  • Department of Computer Science
  • Semester: 04
  • Credits: 4
  • Coefficient: 2

Teaching Objectives

The ultimate goal of this course is to teach students how to implement a Web application.

Recommended Prerequisites

Students should have fundamental concepts on algorithms and programming, along with basic knowledge about the Internet and Networks.

Chapter 1: Introduction to the World Wide Web

  • Definition and history
  • Client/Server Architecture
  • HTTP Protocol

Chapter 2: Programming Languages for the Web

Generalities

  • Static page, dynamic page, and Web applications

Markup Languages

  • HTML
  • CSS
  • JavaScript
  • XML

Chapter 3: Server-Side Programming Language (PHP)

  • Introduction
  • Basic Syntax
  • Types, Variables, and Operators
  • Control Structures
  • Classes and Objects
  • Features

Chapter 4: Web Services: Basic Concepts

  • Introduction
  • Service-Oriented Architecture (SOA)
  • Web Services Features
  • Standards for Web Services

Chapter 5: Case Study: Development of a Web Service (provider side then client side)

Evaluation Mode

  • Examination: 60%
  • Continuous Assessment: 40%
