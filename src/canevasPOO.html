Object-Oriented Programming

  • Department of Computer Science
  • Semester: 4
  • Credits : 4
  • Coefficient: 2

Teaching Objectives

  • Understanding the essence of object-oriented programming and its translation into the Java language.
  • Developing intuitive reasoning to provide a solution to a simple problem using the object-oriented approach.
  • Writing a functional program in the Java language.
  • Understanding the essence and importance of object-oriented reasoning and OOP.

Prerequisite Knowledge

Language C
